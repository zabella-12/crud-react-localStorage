# Crud React localStorage

<p>Cadastro básico de nome e e-mail, que será salvo no localStorage, com operações básicas de:</p>
<ul>
<li>create;</li>
<li>update;</li>
<li>deleted;</li>
<li>read.</li>
</ul>


## Descrição do Projeto

Utilizado [React](https://react.dev/learn) com [Chakra UI](https://chakra-ui.com/getting-started). Para execução do projeto após o download, é necessário ter instalado [NodeJs](https://nodejs.org/en)


Para startar, execute o comando no terminal raíz do projeto:
<code>yarn dev</code>

Tutorial reproduzido de Will Dev- [Vídeo](https://www.youtube.com/watch?v=FqbgZ3lwGnM)
